How to delete a repository from Github 2013-03-04

click and go the repository to be deleted.
then click the “settings”

or github.com/username/repository_name/settings

there is a “denger zone” pannel at the end of the page.
find “delete this repository” and click “delete this repository”
follow the instruction.

reference: github如何删除一个repository(仓库)
note: this is obsolte now, since new version of github is deployed.
